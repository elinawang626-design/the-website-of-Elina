# Elina Personal Profile

A lightweight personal profile website styled as a Jump Jump-inspired experience map.

The page presents Yining Wang / Elina's application narrative around trustworthy, verifiable, and deployable AI systems for health and biomedical information settings.

## Local Preview

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 4173
```

Then visit `http://127.0.0.1:4173/`.
