# Elina Personal Profile

A React + Vite + TypeScript personal profile website styled as a Jump Jump-inspired experience map.

The page presents Yining Wang / Elina's application narrative around trustworthy, verifiable, and deployable AI systems for health and biomedical information settings.

## Local Preview

```bash
npm install
npm run dev
```

Then visit the Vite local URL, usually `http://127.0.0.1:5173/`.

## Build

```bash
npm run build
```
